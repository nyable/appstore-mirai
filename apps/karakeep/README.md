## 介绍

Karakeep（前身为 Hoarder）是一款开源的“书签一切”应用，它利用人工智能自动为您存储的内容打上标签。该应用优先支持自托管部署。

## 功能特性

- 🔗 收藏链接、记录简单笔记、存储图片和 PDF 文件。
- ⬇️ 自动抓取链接的标题、描述和图片。
- 📋 将您的书签整理到列表中。
- 🔎 对所有存储的内容进行全文搜索。
  ✨ 基于人工智能（即 ChatGPT）的自动标签生成和内容摘要。支持使用 \* Ollama 的本地模型！
- 🤖 基于规则的引擎，用于定制化管理。
- 🎆 使用 OCR 从图片中提取文本。
- 🔖 Chrome 浏览器插件 和 Firefox 浏览器附加组件，方便快速收藏书签。
- 📱 提供 iOS 应用 和 Android 应用。
- 📰 从 RSS 源自动抓取内容。
- 🔌 提供 REST API 和多个客户端。
- 🌐 支持多语言。
- 🖍️ 标记并存储您收藏内容中的重点内容。
- 🗄️ 完整页面归档（使用 monolith），防止链接失效。
- ▶️ 使用 yt-dlp 自动归档视频。
- ☑️ 支持批量操作。
- 🔐 支持 SSO（单点登录）。
- 🌙 支持深色模式。
- 💾 优先支持自托管部署。
- ⬇️ 支持从 Chrome、Pocket、Linkwarden、Omnivore、Tab Session Manager 导入书签。
- 🔄 通过 floccus 与浏览器书签自动同步。
