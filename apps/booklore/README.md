## 介绍

BookLore 是一个自托管的网络应用程序，用于组织和管理您的个人图书收藏。它提供了一个直观的界面，用于浏览、阅读和跟踪您的 PDF 和电子书的进展。通过强大的元数据管理、多用户支持和时尚现代的用户界面，BookLore 使构建和探索您的个人图书馆变得简单。

## 功能特性

- 📚 智能组织 ：使用自定义书架、智能分类和强大的筛选功能，打造你的梦想图书馆，轻松找到任何一本书。
- 👥 多用户管理 ：为图书馆访问和内容控制添加具有细粒度权限的用户
- 📲 Kobo 集成 ：与 Kobo 设备无缝同步您的图书馆，并自动将 EPUB 转换为 KEPUB
- ✨ 魔法货架 ：创建动态的基于规则的智能收藏，实时更新 - 类似于书籍的智能播放列表
- 🧠 自动元数据 ：从 Goodreads、Amazon、Google Books 和 Hardcover 上获取丰富的图书详情
- 📤 BookDrop 导入 ：将文件放入文件夹中进行自动检测和批量导入
- 🌐 OPDS 支持 ：直接将阅读应用程序连接到您的图书馆，进行无线下载。
- 🔑 灵活的身份验证 ：选择本地账户或外部 OIDC 提供商（Authentik，Pocket ID）
- 🔄 KOReader 同步 ：在 KOReader 和 BookLore 之间跟踪阅读进度
- 📧 一键分享 ：直接从界面发送图书的电子邮件
- 🔐 私人笔记 ：保存个人阅读笔记，仅对您可见
- 🌍 社区评论 ：自动获取公共评论以丰富您的图书馆
- 📖 内置阅读器 ：使用可定制的主题阅读 PDF、EPUB 和漫画
- 📱 移动设备准备就绪 ：完全响应式设计，适用于所有设备
